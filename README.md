# mgc
Tool for scraping MyCamu timetable for common calendar app usage

## Project Name: MGC (Mycamu is not a Good Calendar)

## What does it do
MGC is a project to scrape a student’s timetable from MyCamu and place events in one’s preferred calendar app. It will use web techologies and operate as a Chrome extension.

## MVP (Minimum Viable Product) execution
1.	The extension should ensure the user is signed in to the website’s (MyCamu). (Have the user go to Camu, if they are not on the profile or similar and on login, make them login to proceed)
2.	The extension should request the time period for their timetable.
3.	The website should rely on Camu v2. (checks with v1 and latter ones can be added later)
4.	An option to generate an ics file should be given to click.
5.	An ics file should be generated and a guide on how to use in apps should be provided. (When clicked, the extension should go to the timetable path of the website, perform some actions, scrape data and process it to ics)

### Extra considerations
1.	The user should then click on a button for their preferred calendar apps. 
2.	Login details for the calendar app should be provided
3.	Calendars of importance are Microsoft Outlook and Google calendar.
